# Personal Academic Homepage

Simple GitHub Pages template for an academic personal homepage.

## Files

- `index.html`: main page
- `style.css`: page styles
- `assets/profile-placeholder.svg`: default profile image placeholder

## Quick customize

1. Replace avatar:
   - put your photo at `assets/profile.jpg`
   - update `index.html` image source from `assets/profile-placeholder.svg` to `assets/profile.jpg`
2. Add your CV:
   - put PDF at `assets/Yuping_Qiu_CV.pdf`
3. Update publication list:
   - edit the `Publications` section in `index.html`
4. Update contact email:
   - edit the `Contact` section in `index.html`

## Deploy to GitHub Pages

1. Create or use repo `username.github.io`.
2. Push these files to the repository root.
3. Enable GitHub Pages (if not auto-enabled).
4. Visit `https://username.github.io`.
